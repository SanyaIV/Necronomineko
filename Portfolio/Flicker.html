Flicker

This was the fourth game I made. It was made by me and four other students during the course "Project work in Game design (SP:PROJ)" at Stockholm University.

My responsibilities for this project were coding, level building (based on design), connecting assets to code and managing the Git repository and being support for all the git issues we encountered as well as some other smaller parts.

  • Engine: Unity3D

  • Main Programming Language: C#

  • Platform: Windows (Primary)

  • Source Control: GitHub for Unity - GitHub Repository.

  • Download: Windows 64bit
